Splash Page

The splash screen is a visually engaging introduction to the website, featuring a library background, a mission statement, and a progress bar. It highlights the project’s goal of promoting quality education.


HTML Techniques

The Splash Page is structured using key HTML5 elements for a visually engaging introduction. The <div class="splash-screen"> tag wraps the content, which includes the main heading, mission statement, and list of group members. The <h1> tag is used to display the primary heading, while <p> tags provide the mission statement and additional information. The <ul> and <li> tags are used to list the group members. A loading bar is created using <div class="loading-bar"> and <div class="progress">, and the <script> tag handles the automatic redirection to the home page after a few seconds.


Key HTML elements include:

  • <div class="splash-screen">: Wraps the splash content, providing a structured layout.
  • <h1>: Displays the main heading, setting the theme of the page.
  • <p>: Used for the mission statement and group member section, ensuring clarity.
  • <ul> & <li>: Lists group members in an organized manner.
  • <div class="loading-bar"> & <div class="progress">: Creates a visual loading bar that animates before redirecting.
  • <script>: Controls the automatic redirection to the home page after 4 seconds.


CSS Techniques

The Splash Page is styled using modern CSS techniques to create an attractive and responsive layout. The page utilizes a full-screen background image to capture the user’s attention, and the content is centered with padding and margins to ensure clarity and focus.

Key CSS properties include:

  • Background Image: The background property sets a full-screen background image, making the page visually appealing and immersive.
  • Flexbox: The text-align: center property centers the text and content, creating a balanced and visually pleasant layout.
  • RGBA Color: The background-color property uses an RGBA value to create a semi-transparent overlay that enhances readability against the background image.
  • Font Size and Spacing: font-size and padding are applied to ensure that text is readable and well-spaced, with distinct sections for the heading, mission statement, and list.
  • Loading Bar: The .loading-bar and .progress classes are used to create a visually appealing loading bar with smooth animation using the transition property.
  • Border Radius: Applied to elements like the loading bar and progress indicator to create rounded corners, giving the page a modern look.


Challenges and Lessons Learned

During the development of the splash Page, one challenge was ensuring the layout remained consistent across different screen sizes. This was resolved by thoroughly testing the page on multiple devices and adjusting the media queries as needed. Another challenge was balancing visual appeal with performance, which was addressed by optimizing image sizes and minimizing CSS code. These experiences highlighted the importance of responsive design and performance optimization in web development.



Compliance

The Splash Page complies with web standards and accessibility guidelines. It uses semantic HTML5 elements to ensure a clear structure, with descriptive alt text for any images (such as the background). The page has been validated using the W3C Validator, ensuring it meets industry standards for HTML and CSS. Additionally, the design focuses on accessibility, with good contrast for readability and an intuitive layout for a positive user experience.

Volunteer Page

The Volunteer Page is a dedicated section of the website aimed at promoting and highlighting various volunteer opportunities. This page features a collection of ongoing and upcoming projects related to quality education and empowerment. Users can explore details about different programs, view the project's description, and get involved by clicking on the "VIEW PROJECT" button. The page also includes ratings, reviews from participants, and a feedback section, encouraging community engagement and interaction. The purpose is to inspire visitors to take part in meaningful volunteer work and contribute towards the mission of providing quality education.



HTML Techniques

The Volunteer Page uses semantic HTML5 tags to ensure structure, clarity, and accessibility. The <header> tag contains the navigation menu, linking to important sections such as Home, Volunteer, and Profile Pages. The main content is divided into meaningful sections using <section> tags, each highlighting key volunteer opportunities. <article> tags are employed for individual program cards, which present project details. The <footer> section includes links to the editor's page, team members, and other navigational elements.

Key HTML elements include:

  • <h1> and <h2> tags for headings, creating a well-defined content hierarchy.
  • <p> tags for descriptive text, ensuring the content is engaging and informative.
  • <section> tags to group relevant content such as volunteer programs and ratings.
  • <article> tags for individual program cards that display detailed project information.
  • <a> tags for navigation, making it easy for users to explore different volunteer projects.
  • <footer> for quick links, team information, and credits.

CSS Techniques

The Volunteer Page is styled using modern CSS techniques to create an engaging and responsive design. Flexbox is used to center elements like the navigation menu and footer, while Grid is employed to organize volunteer program cards and ratings sections. The page follows a consistent color scheme, typography, and spacing that enhances readability and aligns with the overall theme of the website.

Key CSS properties include:

  • Flexbox: Centers the navigation menu and footer content, ensuring a clean and responsive layout across devices.
  • Grid: Organizes the volunteer program cards into a visually appealing layout, adapting to different screen sizes.
  • Media Queries: Ensures the page is fully responsive, adjusting the layout for desktops, tablets, and mobile devices.
  • Transitions and Hover Effects: Adds subtle hover effects to buttons and program cards, improving user interaction.
  • Box Shadow and Border Radius: Gives depth to program cards and buttons, creating a modern look and feel.
  • Typography: Utilizes clear and legible fonts, enhancing readability and aligning with the theme of education and volunteerism.


Purpose of Key Tags and Styling

  • section: Used to organize different parts of the volunteer page, such as the program details, ratings, and feedback sections, creating a clear structure.
  • Flexbox and Grid: Ensure a visually appealing layout for the volunteer program cards and ratings, while maintaining responsiveness across different screen sizes.
  • Media Queries: Guarantee the page is fully responsive, adjusting the layout for desktops, tablets, and mobile devices, ensuring an optimal experience for all users.
  • Semantic HTML Tags: Enhance accessibility and SEO by providing meaningful structure, making it easier for users and search engines to navigate and understand the page content.
  • button: Used for interactive elements like "VIEW PROJECT" and "Submit Feedback" to engage users in the volunteer process and ensure smooth navigation.




Accessibility Features

The Volunteer Page is designed with accessibility in mind, ensuring an inclusive experience for all users.

  • Descriptive alt text for all images, ensuring screen readers can convey the content to visually impaired users.
  • High contrast ratios between text and background colors to enhance readability for users with visual impairments.
  • ARIA labels for interactive elements, such as buttons and links, to improve navigation for users relying on assistive technologies.
  • Keyboard navigability, enabling users to interact with the page without requiring a mouse.


Challenges and Lessons Learned

One challenge during the development of the Volunteer Page was ensuring that the volunteer program cards were visually appealing and well-structured across different screen sizes. This was resolved by using Flexbox and Grid to align the elements and media queries to optimize the layout for smaller screens.



Compliance

The Volunteer Page complies with Janet regulations and web standards by using semantic HTML5 tags, ensuring clarity and accessibility. It includes descriptive alt text for images, making the content more inclusive for users with disabilities. The page adheres to accessibility guidelines and follows best practices for responsive design. Additionally, it has been validated using the W3C Validator, ensuring that it meets industry standards for HTML and CSS for a well-structured and user-friendly experience.
