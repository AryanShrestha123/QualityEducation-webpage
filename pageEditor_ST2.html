Navigation Bar

The Navigation Bar serves as the primary means for users to explore different sections of the website. It is designed to be fixed, ensuring accessibility at all times while maintaining a structured layout.

The Navigation bar is structured using semantic HTML5 tags to ensure clarity and accessibility.

Key Features:

  • Fixed Positioning: Stays visible at the top of the page even when scrolling. of different pages.
  • Active Link Highlighting: The currently active page is visually distinct.
  • Semantic Structure: Improves SEO and accessibility.


Footer

The footer serves as a global navigation and information section at the bottom of every page. It provides:

It displays the editor of the page, link to the respective editor's page, navigation links same as in the header, team members with their editor page.


Key HTML elements:

  • <header>: Wraps the entire navigation structure.
  • <nav>: Contains the navigation links for better accessibility of different pages.
  • <ul> and <li>: Organize menu items in a structured list format.
  • <a>: Provide links to different sections of the website, ensuring seamless navigation.
  • Class Names: Used for targeted styling via CSS.


CSS Techniques

CSS is used to style the layout, colors, typography, and effects. The goal is to create a visually appealing and cohesive design that aligns with our theme and enhances user experience.

The global css file is linked to every single page as it styles the navigation bar, footer, and a little layout for the main section.


Key CSS properties include:

  • Flexbox: Centers the main content and navigation menu, ensuring adaptability to various screen sizes.
  • Media Queries: Makes the page fully responsive for desktops, tablets, and mobile devices.
  • Transitions and Animations: Adds subtle hover effects and smooth scroll behaviour for enhanced user interaction.
  • Box Shadow and Border Radius: Creates depth and a modern look for cards and buttons.


Purpose of Key Tags and Styling

  • <header> and <nav>: Provide a clear, accessible navigation system for an enhanced user experience.
  • Flexbox: Ensure a responsive, visually appealing layout adaptable to various screen sizes.
  • Media Queries: Enable full responsiveness, ensuring accessibility across all devices.
  • Semantic HTML Tags: Improve accessibility and SEO, making the page easy to navigate and understand.


Accessibility Features

The Home Page is designed with accessibility in mind. Key features include:

  • Descriptive alt text for all images, ensuring screen readers can convey the content to visually impaired users.
  • Proper contrast ratios for text and background colors, making the content readable for users with visual impairments.
  • ARIA labels for interactive elements, such as buttons and links, to improve navigation for users relying on assistive technologies.


Compliance

The website layout complies with Janet regulations and web standards. It uses semantic HTML5 tags, includes descriptive alt text for images, and adheres to accessibility guidelines. The page has been validated using the W3C Validator, ensuring it meets industry standards for HTML and CSS.


Home Page

The Home page is the key page of the website as it introduces users to the core message of the website, providing informative content about Quality Education and Sustainable Development Goal (SDG) 4.

It adapts the global css file and aligns with the theme of the website. It includes introduction, links to content pages in form of cards, a learn more button that scrolls to the commitment section, and what we do section.



Key HTML elements:

  • <main>: Wraps the main content of the page.
  • <section>: Defines different content sections (Intro, Featured Content, Commitment, etc.).
  • <h1>, <h2> and <h2>: Used for headings to establish a clear content hierarchy.
  • <p>: Tags for descriptive text, ensuring readability.
  • <a>: Creates hyperlinks for navigation and interactive elements.
  • <img>: Displays images with descriptive alt attributes for accessibility.
  • <div>: Used for structuring content (e.g., wrapping content cards, buttons, and layout containers).
  • <button>: Used for Learn More button.


CSS Techniques

The Home Page is styled using modern CSS techniques to create a visually appealing and responsive design. The page features a consistent color scheme and typography that aligns with the overall theme of the website.

Key CSS properties include:

  • Flexbox: Used for flexible, responsive layouts.
  • Trasition and transform: Enables smooth animations and hover effects.
  • Media Queries: Ensure the page is fully responsive for desktops, tablets, and mobile devices.
  • Box Shadow and Border Radius: Add depth and a modern look to sections.


Purpose of Key Tags and Styling

  • justify-content & align-items: Aligns elements properly within sections for a clean and professional design.
  • Flexbox: Ensure the layout is visually appealing and functional, adapting to various devices and screen sizes.
  • Pseudo classes, transition & transform: Adds smooth hover effects and interactive animations.
  • Semantic HTML Tags: Improve accessibility and SEO, ensuring the page is easy to navigate and understand.


Accessibility Features:

  • alt attributes in img tags - Provides descriptive text for visually impaired users using screen readers.
  • Proper contrast ratios for text and background colors, making the content readable for users.
  • ARIA labels - Enhances screen reader interpretation of buttons and links.
  • Use of nav and header - Helps screen readers identify key navigation sections.


Challenges and Lessons Learned

One challenge during the development of the Table Page was ensuring the data tables remained readable and visually appealing on smaller screens. This was resolved by using media queries to adjust the table layout for mobile devices. Another challenge was organizing the large amount of data into a coherent structure, which was addressed by using clear headings and color-coded rows. These experiences emphasized the importance of responsive design and effective data presentation in web development.



Compliance
